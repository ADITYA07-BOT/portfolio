# 🌟 Aditya Choudhary - Portfolio Website

A modern, responsive portfolio website showcasing projects, skills, and professional experience. Built with immersive animations, video backgrounds, and interactive elements.

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://your-domain.com) 
![Tech Stack](https://img.shields.io/badge/Stack-HTML5/CSS3/JS-blue?style=for-the-badge)

![Portfolio Preview](https://via.placeholder.com/800x500.png?text=Portfolio+Showcase)

## 🚀 Features

- **Immersive Visual Design**
  - Gradient backgrounds with dark theme
  - Animated video overlays
  - Floating element animations
  - Smooth scroll navigation

- **Interactive Sections**
  - Project gallery with hover effects
  - Dynamic tab system (Skills/Experience/Education)
  - Music player controls
  - Downloadable CV button

- **Modern Functionality**
  - Google Sheets contact form integration
  - Responsive mobile-first design
  - Social media integration
  - Audio/Video background controls

- **Content Sections**
  - Hero section with animated CTA
  - Project showcase with live demos
  - Professional timeline experience
  - Contact form with submission feedback
